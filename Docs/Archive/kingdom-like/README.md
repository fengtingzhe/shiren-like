# kingdom-like 旧方向归档

本目录用于记录本仓库曾经使用过的 `kingdom-like` 轻策略探索建造 Demo 方向。

自 `2026-05-13 / v0.0 / 项目重定向版` 起，人类制作人已明确决定：

```text
停止继续开发旧 kingdom-like 方向；
以现有仓库结构为基础，制作 shiren-like / Mystery Dungeon Roguelike Demo。
```

---

## 旧方向摘要

旧项目曾经定位为：

```text
一款面向泛用户的轻策略探索建造 Demo：玩家孤身进入森林探索未知，用唯一资源“金币”命令村民砍树、建造和升级，把发现的流民、地标和机会逐步纳入王国安全范围，并在夜晚承受扩张带来的防守压力。
```

旧方向关键词：

- 探索森林；
- 金币单资源；
- 工人砍树；
- 固定节点建造；
- 建墙 / 建塔；
- 王国边界扩张；
- 夜晚防守压力。

---

## 新方向摘要

新项目定位为：

```text
一款现代化 Mystery Dungeon Roguelike Demo：玩家在随机迷宫中逐层探索，以回合制格子移动、道具组合、怪物机制、陷阱、饥饿压力和死亡复盘为核心，体验“每一步都是决策”的高密度迷宫冒险。
```

新方向关键词：

- 格子迷宫；
- 回合制；
- 怪物追击；
- 道具解局；
- 楼梯下楼；
- 饥饿压力；
- 陷阱；
- 未鉴定；
- 死亡复盘。

---

## 后续使用规则

- 旧方向只作为历史记录和参考；
- 后续 AI 不应继续开发旧 kingdom-like 的玩法系统；
- 如需要查看旧内容，请通过 Git 历史或 v0.0 前的提交记录查看；
- 当前项目的设计来源以 `README.md` 和 `DESIGN_HUB/` 中的 shiren-like 文档为准。
