# shiren-like Web Demo

当前版本：`v0.6 / 陷阱与危险地形版`

这一版的目标，是让地图本身开始参与战术压力。玩家不再只需要处理怪物，还要观察地面、规避陷阱，并在走位时考虑持续危险地形。

## 已有系统

- 随机房间与走廊
- 玩家中心镜头
- `Console / Camera / View` 调参
- 饥饿与饱腹度
- 小地图、敌人面板、底部快捷栏
- 恢复药、食物、传送卷轴、睡眠卷轴、火球杖、换位杖
- 四种怪物：
  - `Slime`
  - `Goblin Archer`
  - `Sleep Mushroom`
  - `Skeleton Spearman`

## v0.6 新增内容

### 陷阱

- `Spike Trap`
  - 玩家踩上去后立即受到固定伤害
  - 单次触发后失效
- `Warp Trap`
  - 玩家踩上去后被传送到当前层安全地板
  - 不会把玩家送到墙、怪物、楼梯、道具、其他陷阱或危险地形上
  - 单次触发后失效
- `Sleep Trap`
  - 玩家踩上去后进入睡眠
  - 复用 `v0.5` 的 `player.sleepTurns`
  - 单次触发后失效

### 危险地形

- `Poison Pool`
  - 进入毒沼时受到 1 点伤害
  - 站在毒沼上等待、攻击或使用道具时也会继续受到 1 点伤害
  - 不会消失

## 生成规则

- 陷阱和危险地形都只生成在地板上
- 不生成在玩家出生点、楼梯、楼梯相邻 1 格上
- 不与怪物、道具重叠
- 为了避免把唯一通道堵成必踩危险，当前优先把它们放在房间里，而不是走廊里
- `1F` 数量较少，`2F` 到 `3F` 逐步增加

## 显示策略

- 主画面中：
  - 地刺陷阱显示为红色尖刺
  - 传送陷阱显示为紫色问号
  - 睡眠陷阱显示为蓝紫色 `Z`
  - 毒沼显示为绿色半透明地面覆盖
- 小地图当前只显示持续危险地形 `Poison Pool`，不显示一次性陷阱，避免信息过密

## 回合规则

- 踩中陷阱属于本次行动的一部分，不会额外多推进一回合
- 陷阱触发后，若玩家未死亡，仍然只推进正常的一次怪物行动
- 毒沼伤害会在玩家行动结算阶段触发，所以：
  - 走进毒沼会掉血
  - 站在毒沼上等待或使用道具也会掉血
- 睡眠陷阱复用已有睡眠逻辑，后续回合自动跳过玩家行动

## 运行

在项目根目录运行：

```text
node Tools/web-demo-server.mjs
```

然后打开：

```text
http://127.0.0.1:4173/Builds/web-demo/
```

也可以直接运行：

```text
Tools/run-web-demo-vite.bat
```

默认会打开：

```text
http://127.0.0.1:5173/Builds/web-demo/
```
