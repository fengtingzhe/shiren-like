# shiren-like Web Demo

当前版本：`v0.3 / 随机迷宫与饥饿压力版`

本版本用于验证：

```text
随机探索 + 轻量饥饿压力 + 搜刮与下楼取舍
```

玩家每层都会进入重新生成的房间与通道迷宫。继续绕路搜刮会消耗满腹度，快速下楼更安全但可能少拿关键道具。

---

## 运行方式

在项目根目录运行：

```text
node Tools/web-demo-server.mjs
```

然后打开：

```text
http://127.0.0.1:4173/Builds/web-demo/
```

也可以双击：

```text
Tools/run-web-demo.bat
```

---

## 随机迷宫

v0.3 使用简单房间 / 走廊生成：

- 地图尺寸为 15×12；
- 每层生成 4～6 个矩形房间；
- 按房间中心连接走廊；
- 玩家出生在第一个房间；
- 楼梯放在远离玩家的房间；
- 怪物和道具随机放在合法地板格；
- 若随机生成失败，会回退到固定备用地图。

每次下楼都会重新生成当前层地图、怪物和道具分布。

---

## 满腹度 / 饥饿

- 初始满腹度：100 / 100；
- 每 5 个有效回合，满腹度下降 1；
- 满腹度低于 25 时，HUD 变红并写入日志提示；
- 满腹度为 0 后，每 5 个有效回合受到 1 点饥饿伤害；
- 饥饿伤害可以导致死亡，结算会显示饥饿死因；
- 食物恢复 50 点满腹度，成功使用后推进回合。

有效回合包括移动、普通攻击、成功使用道具、等待和下楼。撞墙、没有道具、HP 已满无法喝药、满腹已满无法吃食物、打开 Console、暂停都不会消耗满腹度。

## 保留的 v0.2 道具

| 道具 | 快捷键 | 作用 |
|---|---|---|
| 回复药 | H | 恢复 HP，成功使用后推进回合 |
| 食物 | F | 恢复满腹度 |
| 传送卷轴 | T | 随机传送到当前楼层安全地板 |
| 睡眠卷轴 | Z | 让 4 格内怪物睡眠 3 回合 |
| 火球杖 | R | 朝当前朝向发射火球，命中第一个怪物，未命中不消耗 |
| 换位杖 | X | 与当前朝向第一个怪物交换位置，未命中不消耗 |

移动仍使用 WASD / 方向键。`S` 保留为向下移动，因此睡眠卷轴使用 `Z`。

## 试玩重点

- 每层地图变化是否足够清楚；
- 玩家是否能理解房间、走廊、楼梯目标；
- 满腹度下降是否能形成“继续搜刮 vs 快速下楼”的取舍；
- 食物数量是否足够但不过量；
- v0.2 的传送、睡眠、火球、换位是否仍能在随机地图中发挥作用。
