# shiren-like Web Demo

当前版本：`v1.0 / 单局体验整合与可读性打磨版`

v1.0 的目标是把现有系统打磨成一个可以稳定试玩、容易理解、反馈清晰的单局体验。本轮未加入任何大系统（壶、商店、怪物屋、据点、Boss、Godot）。

## 操作说明

- 移动：WASD / 方向键
- 等待：空格 / .
- 下楼：E / Enter
- 装备脚下装备：C
- 收入背包：G
- 打开背包：B
- 快捷使用：H（恢复药）F（食物）T/Z（卷轴）R/X（木杖）
- Console：调试镜头与 FPS

## HUD 信息

HUD 显示：
- HP（低于 5 时变红）
- 饱腹度（低于 25 时变红）
- 当前楼层
- 回合数
- 背包容量
- ATK / DEF
- 当前武器 / 盾牌
- 睡眠状态（睡眠时显示剩余回合）

## 背包面板

- `B` 打开 / 关闭背包
- 消耗品显示名称、数量和未识别标记
- 装备显示名称和数值加成
- 可使用、装备或投掷背包内物品

## 怪物、陷阱与危险提示

- 敌人面板显示距离、行为标签（近战/远程/催眠/长枪）和 HP 条
- 陷阱和毒沼触发时在日志中清晰提示
- 未识别卷轴和木杖使用或投掷后自动鉴定，日志显示三项信息：尝试了什么、发生了什么、鉴定成了什么

## 死亡与胜利结算

结算界面显示：
- 到达楼层
- 总回合数
- 击败怪物数
- 使用道具次数
- 已鉴定道具数量
- 获得装备数量
- 死亡原因（被怪物击败/饥饿/地刺陷阱/毒沼）

## 背包规则

### 未鉴定范围

- 会未鉴定：
  - 传送卷轴
  - 睡眠卷轴
  - 火球杖
  - 换位杖
- 始终已知：
  - 恢复药
  - 食物
  - 所有装备

### 识别规则

- 每局开始时，未鉴定道具会随机分配一组未知名称
- 同一局内，这组未知名称保持固定
- 使用成功后，会识别该类型道具
- 投掷后只要完成结算，也会识别该类型道具
- `火球杖 / 换位杖` 若前方没有目标，当前仍然不消耗，因此也不会识别

示例：

```text
褪色卷轴 x1
黑曜木杖 x1
```

识别后会变为：

```text
传送卷轴 x1
火球杖 x1
```

### 容量

- 背包总容量：`8`
- HUD 会显示：`背包 5 / 8`

### 占格规则

- 同类型消耗品如果数量大于 `0`，只占 `1` 格
- 每件未装备装备占 `1` 格
- 已装备的 `weapon / shield` 不占背包格

示例：

```text
恢复药 x2：1 格
食物 x1：1 格
传送卷轴 x1：1 格
火焰剑备用：1 格
铁盾备用：1 格
总占用：5 / 8
```

## 拾取规则

### 消耗品

- 走上去仍然会自动拾取
- 如果背包已满，无法拾取，物品会留在地上
- 如果背包里已有同类型消耗品，因为允许堆叠，仍然可以继续拾取

### 装备

- `C`：直接装备脚下装备
- `G`：把脚下装备收进背包
- 如果背包已满，`G` 会失败并提示
- 地面 `C` 装备不受背包容量限制
- 地面换装时，旧装备仍然掉回地上，不自动进包

## 背包面板

- `B`：打开 / 关闭背包
- 面板会显示：
  - 当前容量
  - 物品名称
  - 数量
  - 装备数值
  - 未识别状态

示例：

```text
1. 恢复药 x2           [U 使用] [T 投掷]
2. 褪色卷轴 x1 · 未识别 [U 使用] [T 投掷]
3. 火焰剑 +4           [E 装备] [T 投掷]
```

## 从背包使用与装备

- 快捷栏仍然保留
- 背包面板里也可以使用消耗品
- 使用成功会推进 `1` 回合
- 使用失败不会推进回合
- 背包里的装备可以直接 `E 装备`

从背包装备时：

- 新装备进入对应装备位
- 原装备优先回到背包
- 如果背包放不下，原装备会掉到脚下或相邻安全格
- 如果附近也没有安全格，原装备会消失并写日志

## 投掷规则

- 从背包面板点击 `T 投掷`
- 投掷方向使用玩家当前 `facing`
- 从玩家前方一格开始直线飞行
- 最大距离 `5` 格
- 遇到墙停止
- 命中第一个怪物就结算
- 未命中时：
  - 装备会尝试落在最后一个合法地板格
  - 其他投掷物会消失
- 未识别卷轴或杖在投掷结算后会识别

## 投掷效果

- 恢复药：`2` 点伤害
- 食物：`1` 点伤害
- 传送卷轴：`2` 点伤害
- 睡眠卷轴：让命中怪物睡眠 `3` 回合
- 火球杖：`3` 点伤害
- 换位杖：`2` 点伤害
- 武器：`2 + attackBonus`
- 盾牌：`1 + defenseBonus`

关键规则：

- 投掷会从背包中消耗物品
- 命中怪物后投掷物消失
- 已装备物品不参与投掷，只有背包里的物品可投掷

## 运行

在项目根目录运行：

```text
node Tools/web-demo-server.mjs
```

然后打开：

```text
http://127.0.0.1:4173/Builds/web-demo/
```

也可以直接运行：

```text
Tools/run-web-demo-vite.bat
```

默认会打开：

```text
http://127.0.0.1:5173/Builds/web-demo/
```
