# shiren-like Web Demo

当前版本：`v0.5 / 怪物机制版`

这一版的目标不是继续扩系统，而是让怪物开始具备明确的处理差异。现在 Web Demo 里已经有第一批四种怪物：

- `Slime`：基础近战怪，贴身攻击。
- `Goblin Archer`：同一行或同一列、4 格内、无墙阻挡时进行远程射击；贴身时也能近战。
- `Sleep Mushroom`：2 格曼哈顿范围内释放催眠孢子，让玩家睡眠 2 回合，技能冷却 4 回合。
- `Skeleton Spearman`：同一行或同一列、1 到 2 格内、无墙阻挡时进行隔格刺击。

主画面保留了 `v0.4.3` 的玩家中心镜头和 `Console / Camera / View` 调参能力；随机房间、饥饿、3 层通关、右上小地图、右侧敌人面板、底部快捷栏，以及恢复药、食物、传送卷轴、睡眠卷轴、火球杖、换位杖都继续可用。

## 运行

在项目根目录运行：

```text
node Tools/web-demo-server.mjs
```

然后打开：

```text
http://127.0.0.1:4173/Builds/web-demo/
```

也可以直接运行：

```text
Tools/run-web-demo-vite.bat
```

默认会打开：

```text
http://127.0.0.1:5173/Builds/web-demo/
```

## v0.5 本轮改动

- 升级为多怪物配置，`Data/config/web_demo_balance.json` 新增 `monsters` 字段。
- 每层随机怪物池按 `monsterTypes` 权重生成。
- 保留按房间分配怪物，每个非出生房间至少 1 只怪，大房间额外加压。
- 新增玩家 `sleepTurns`，睡眠期间输入会自动跳过回合，怪物继续行动。
- 主画面威胁格不再一律只画近战相邻格，而是按怪物行为显示：
  - 近战：相邻 4 格
  - 弓手：直线射程
  - 蘑菇：2 格催眠范围
  - 枪兵：直线 2 格
- 右侧敌人面板现在显示怪物图标、颜色、HP 和状态/距离/冷却。
- 怪物渲染改成按类型着色，并显示各自字母图标。

## 楼层怪物分布

- `1F`：以 `Slime` 为主，少量 `Goblin Archer`
- `2F`：加入 `Sleep Mushroom`
- `3F`：四种怪物都可能出现，`Skeleton Spearman` 作为更高威胁单位登场

## 道具兼容

`v0.2 / v0.3` 的道具仍然全部保留，并且对新怪物生效：

- `恢复药`：回血
- `食物`：恢复饱腹度
- `传送卷轴`：脱离远程压制或包围
- `睡眠卷轴`：群体睡眠所有类型怪物
- `火球杖`：远程击伤所有类型怪物
- `换位杖`：与任意类型怪物交换位置
