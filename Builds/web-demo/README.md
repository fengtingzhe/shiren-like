# shiren-like Web Demo

当前版本：`v0.2 / 道具策略版`

本版本用于验证：

```text
道具可以改变战局。
```

玩家不再只能靠普通攻击和回复药处理危机，而是可以用卷轴和杖逃生、控场、远程攻击或换位解局。

---

## 运行方式

在项目根目录运行：

```text
node Tools/web-demo-server.mjs
```

然后打开：

```text
http://127.0.0.1:4173/Builds/web-demo/
```

也可以双击：

```text
Tools/run-web-demo.bat
```

---

## 已实现内容

- 15×12 小型固定迷宫，共 3 层；
- 墙、地板、玩家、怪物、多种道具、楼梯；
- 玩家有效行动后推进 1 回合，怪物行动 1 次；
- 怪物追击、贴身攻击、睡眠状态；
- 玩家普通攻击、等待、下楼；
- 快捷道具栏和道具数量显示；
- 食物与满腹度占位 UI；
- 传送、睡眠、火球、换位等基础解局道具；
- 胜利 / 死亡结算；
- 通用 Console：音乐开关、音效开关、暂停 / 继续、重置 Demo、显示 FPS。

## 道具与快捷键

| 道具 | 快捷键 | 作用 |
|---|---|---|
| 回复药 | H | 恢复 HP，成功使用后推进回合 |
| 食物 | F | 恢复满腹度；v0.2 不启用饥饿死亡 |
| 传送卷轴 | T | 随机传送到当前楼层安全地板，成功使用后推进回合 |
| 睡眠卷轴 | Z | 让 4 格内怪物睡眠 3 回合，成功使用后推进回合 |
| 火球杖 | R | 朝当前朝向发射火球，命中第一个怪物，未命中不消耗 |
| 换位杖 | X | 与当前朝向第一个怪物交换位置，未命中不消耗 |

移动仍使用 WASD / 方向键。`S` 保留为向下移动，因此睡眠卷轴使用 `Z`。

## 试玩重点

- 道具是否真的能帮助玩家脱离危险；
- 传送卷轴是否能作为逃生手段；
- 睡眠卷轴是否能制造控场窗口；
- 火球杖是否能在怪物接近前处理威胁；
- 换位杖是否能带来位置解局；
- 快捷道具栏是否清楚显示数量、快捷键和用途。
