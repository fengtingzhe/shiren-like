# shiren-like Web Demo

当前版本：`v0.1 / 最小可玩迷宫版`

本版本用于验证最小 Mystery Dungeon 闭环：

```text
格子迷宫 → 回合制移动 → 怪物追击 → 普通攻击 → 回复药 → 楼梯下楼 → 简单胜负
```

---

## 运行方式

在项目根目录运行：

```text
node Tools/web-demo-server.mjs
```

然后打开：

```text
http://127.0.0.1:4173/Builds/web-demo/
```

也可以双击：

```text
Tools/run-web-demo.bat
```

---

## 已实现内容

- 15×12 小型固定迷宫；
- 墙、地板、玩家、怪物、回复药、楼梯；
- WASD / 方向键格子移动；
- 玩家每次有效行动推进 1 回合；
- 怪物每回合追击玩家，贴身时攻击玩家；
- 玩家向相邻怪物移动时执行普通攻击；
- 回复药拾取、HUD 显示、快捷按钮 / H 键使用；
- 站在楼梯上按 E / Enter 或按钮下楼；
- 到达第 3 层终点后胜利；
- HP 为 0 时失败；
- 结算界面显示到达楼层、击败怪物、使用回复药数量；
- 通用 Console：音乐开关、音效开关、暂停 / 继续、重置 Demo、显示 FPS。

---

## 试玩重点

- 是否能理解“玩家行动一次，怪物行动一次”；
- 是否能理解攻击、等待、用药和下楼；
- HP、楼层、回合、回复药数量是否清楚；
- 行动日志是否能解释刚才发生的事；
- 死亡或胜利是否能被玩家理解。
