# NEXT_CODEX_PROMPT: 下一轮 Codex 任务

## 任务名称

Web Demo v0.5 / 怪物机制版

## 目标

在 v0.4.2 的房间视野和房间怪物分配基础上，开始补第一批真正区分打法的怪物机制，但仍然保持 Web Demo 轻量、可试玩、无复杂系统膨胀。

## 建议范围

- 增加 2 到 3 种行为差异明确的怪物
- 保持现有基础怪继续可用
- 优先考虑：
  - 远程压制怪
  - 睡眠/干扰怪
  - 隔格攻击怪
- 保留当前 HUD、小地图、敌人面板、快捷栏、饥饿和基础道具

## 本轮不做

- 不做装备、未鉴定、壶、商店、据点、Boss
- 不做 Godot 工程
- 不推翻 v0.4.2 的房间视野镜头方案

## 开始前要点

先试玩并确认 v0.4.2 的房间镜头、房间怪物密度和走廊跟随是否手感正确，再决定 v0.5 第一批怪物的具体组合与数值。
